# Chat-App

# Download the Code:
git clone https://github.com/Pratik8418/chat_app.git

# Install dependencies using npm:

"dependencies": {
    "bad-words": "^3.0.0",
    "express": "^4.16.4",
    "nodemon": "^1.19.4",
    "socket.io": "^2.5.0"
  }
 
-> npm run dev : 
        "nodemon src/index.js"
 
-> Now Open a browser and enter this url : http://127.0.0.1:8000/
